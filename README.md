# Personal Portfolio Website

A responsive, high-performance personal portfolio website built to showcase my skills, experience, and projects in AI/ML, Data Analytics, and Full-Stack Development. Designed with a premium dark glassmorphism aesthetic.

## 🛠️ Built With
- **HTML5** & **CSS3** (Vanilla UI styling)
- **Vanilla JavaScript** (DOM manipulation, Intersection Observer animations)
- **FontAwesome** (Icons)
- **Google Fonts** ('Inter' and 'Outfit')
- **Formspree** (Contact Form integration)

## ✨ Features
- **Responsive Layout:** fully optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
- **Modern Aesthetics:** elegant dark theme, subtle glowing neon accents, and sleek glassmorphism card designs.
- **Scroll Animations:** clean, staggered reveal animations triggered as the user scrolls down the page.
- **Functional Contact Form:** integrated with Formspree to seamlessly receive messages right in my inbox.
- **Dynamic Projects Section:** quick links to my active GitHub repositories and live project dashboards.

## 🚀 Getting Started Locally

To run this project on your local machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/Varshini-rupa/Varshini-rupa.github.io.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Varshini-rupa.github.io
   ```
3. Open `index.html` in any modern web browser or use VS Code's **Live Server** extension for real-time updates.
